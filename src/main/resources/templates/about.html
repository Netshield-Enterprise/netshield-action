About

This is a test application for NetShield secrets detection.

What This App Does

  • Demonstrates Spring Boot web application structure
  • Provides REST API endpoints for testing
  • Uses Thymeleaf for server-side rendering
  • Configured to test NetShield secrets detection
  • Shows proper configuration management with environment variables

Purpose: This application exists to test NetShield's ability to detect hardcoded secrets in Java/Spring Boot projects. When you add secrets to configuration files or code, NetShield should catch them in pull requests.
